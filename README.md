# معرفی

قالب مدیریتی ویولت برپایه ویوجی اس و تیلویند پیاده سازی شده و در دسترس قرار گرفته.

## نصب و راه اندازی

کپی سورس کد

```sh
git clone git@github.com:alpachinof/vuelet.git
```

### نصب وابستگی ها

```sh
npm install
```

### اجرا در محیط لوکال

```sh
npm run dev
```

### بیلد و خروجی گرفتن

```sh
npm run build
```
